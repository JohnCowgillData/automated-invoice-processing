# AutomatedInvoiceProcessing
Automated AP Invoice Processing from PDF to Upload-Ready Excel

This workflow automated vendor-specific AP invoice processing by converting PDF invoices into upload-ready Excel files for the accounting system. A separate supporting automation also generated the files needed to clear and match those invoices, eliminating manual invoice entry for this vendor and creating a rpeatable AP workflow.

Skills and deliverables:

Python, Microsoft Excel, Data Extraction, Automation, Financial Reporting
